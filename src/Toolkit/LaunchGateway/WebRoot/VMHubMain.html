VM Hub Main Page

VM Hub aims to build an open multimedia hub to allow the entire multimedia/vision community in both academia and industry to easily build cloud services and mobile multimedia applications. VM Hub is built on top of a to-be open sourced distributed functional programming platform Prajna (https://github.com/MSRCCS/Prajna), and the VM Hub Client Library and VM Hub App Library are to be open sourced (pending corporate legal review and management approval). With VM Hub, a multimedia researcher or developer can quickly turn his/her single-machine implementation of a multimedia service, e.g., an image/video recognizer, into a service hosted in cloud with multiple service endpoints, with capability to scale by increasing the number of worker ma-chines, balance the load among endpoints, and automatically deal with failure. VM Hub is capable of concurrently support multiple service providers, each of which may support services with different schemas and domains. VM Hub can also support multiple versions of multimedia services on the same schema and domain. This multi-version hosting feature allows the developer to seamlessly test deploy a new implementation of the service without any risk in disrupting the existing service (the old version). VM Hub service endpoints may be simultaneously deployed in a public cloud and / or in a private cluster. We have also provide tools for detailed operating telemetry of the multimedia service endpoints, with capability for developers to customize their telemetry in arbitrary way. The open sourced VM Hub App Library implementation is based on Portable Class Library, and can support mobile Apps across Windows Phone, Android and iOS platforms. We plan to release sample VM Hub Apps for Windows phone, then for Android and iOS platform, all open sourced to allow other multimedia developers to further customize and build their own apps. The VM Hub suite of libraries will greatly reduce the development and deployment effort for the multimedia developers to bring their work to Cloud service, and encourage them to build mobile applications that easily showcase the multimedia service that they have built to a world audience.
